Comandos do Git:

ls  (listar arquivos)
git status
git add .   ou git nomedoarquivo.js
git restore --stage node_modules
git commit -m "escrever um comentario"

git pull --no-rebase origin main
git push origin main  (primeira push)
git push

Em caso de problema:
git push -f origin main
